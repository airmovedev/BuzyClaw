# ClawTower — 产品简介

## 一句话定义

ClawTower 是基于 OpenClaw 开源架构的本地 AI Agent 客户端，让普通用户无需技术背景即可拥有运行在自己 Mac 上的私人 AI 助手。

## 核心理念

**你的 AI，你的数据，你的设备。**

- 🏠 **100% 本地运行** — AI Agent 运行在你的 Mac 上，不依赖任何云服务器
- 🔒 **零数据外传** — 所有对话、记忆、文件存储在本地，不会传输到开发者或任何第三方
- ☁️ **iCloud 私密中转** — macOS 与 iOS 之间的通信完全通过用户自己的 iCloud 账户中转，开发者无法触及
- 🧠 **OpenClaw 驱动** — 底层采用成熟的 OpenClaw 开源架构，社区活跃，持续迭代

## 产品定位

```
OpenClaw（开源项目）
    │
    │  封装
    ▼
ClawTower（App Store 产品）
    │
    ├── macOS App — 内嵌 OpenClaw Runtime，开箱即用
    └── iOS App — 通过 iCloud 远程访问你的 Mac Agent
```

ClawTower **不是**另一个 AI 聊天工具。它是一个让你拥有**真正属于自己的 AI Agent** 的客户端——Agent 住在你的电脑上，能帮你管理文件、查看日历、处理提醒，而且所有数据从不离开你的设备。

## 与竞品的区别

| | ChatGPT / Claude 官方 App | ClawTower |
|---|---|---|
| 数据存储 | 云端（公司服务器） | 本地（你的 Mac） |
| 隐私 | 对话可能用于训练 | 数据不离开设备 |
| 能力 | 纯对话 | 对话 + 操作本地系统（文件/日历/提醒） |
| 记忆 | 有限上下文 | 完整第二大脑，永久记忆 |
| 自定义 | 有限 | 完全可定制（性格/工具/工作流） |
| 多设备 | 云同步 | iCloud 私密中转 |
| 开源 | ❌ | ✅ 基于 OpenClaw |

## 隐私架构

```
┌─────────────────────────────────────────────┐
│              用户的设备生态                    │
│                                               │
│  Mac ──────────────────────── iPhone          │
│  │ ClawTower macOS            │ ClawTower iOS │
│  │ • OpenClaw Runtime         │ • 远程对话     │
│  │ • Agent + 记忆 + 文件       │ • 推送通知     │
│  │ • 全部数据本地存储           │               │
│  │         │                  │               │
│  │         └──── iCloud ──────┘               │
│  │            （用户自己的账户）                 │
│  │                                            │
│  │         │                                  │
│  │         ▼                                  │
│  │    Claude / OpenAI API                     │
│  │    （仅发送当次对话内容，                     │
│  │      不发送历史记忆/文件）                    │
│  │                                            │
└─────────────────────────────────────────────┘

    ❌ 开发者服务器 — 不存在
    ❌ 第三方数据平台 — 不涉及
    ❌ 用户行为追踪 — 不收集
```

**唯一的外部通信：** 用户主动发起对话时，当次消息发送到用户自己选择的 AI 服务（Claude/ChatGPT）获取回复。这与用户直接使用这些服务没有区别。

## 目标用户

1. **隐私敏感用户** — 不想把对话数据交给云端 AI 公司
2. **效率追求者** — 想要一个能操作本地系统的 AI 助手，不只是聊天
3. **苹果生态用户** — Mac + iPhone 用户，习惯 iCloud 生态
4. **AI 好奇者** — 听说过 AI Agent 但不知道怎么用的普通用户

## 商业模式

- **Phase 1：免费** — 用户自备 AI 账号（OAuth 登录），ClawTower 不产生任何费用
- **Phase 2：Freemium + 订阅**（视增长决定）— 基础功能免费，高级功能订阅
