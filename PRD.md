## 1. 产品概述

### 1.1 一句话定义

ClawTower 是基于 OpenClaw 开源架构的本地 AI Agent 客户端，让普通用户无需技术背景即可拥有运行在自己 Mac 上的私人 AI 助手。

### 1.2 核心卖点

- **100% 本地运行** — AI Agent 运行在用户 Mac 上，数据不离开设备
- **零数据外传** — 除用户主动请求 AI 模型时发送必要内容外，不向开发者或第三方上传任何数据
- **iCloud 私密中转** — macOS 与 iOS 之间通过用户自己的 iCloud 账户通信
- **开箱即用** — 内嵌 OpenClaw Runtime，无需终端操作
- **OpenClaw 生态** — 基于成熟开源架构，社区活跃

### 1.3 分发策略

| 平台 | 方式 | 付费渠道 |
|------|------|---------|
| macOS | 官网 .dmg（Developer ID + 公证 + Sparkle 更新） | Stripe / LemonSqueezy |
| iOS | App Store | StoreKit 2 |

## 3. 功能需求

### 3.1 Onboarding & 数据迁移

#### 3.1.1 新用户引导

| 步骤 | 内容 | 必填 |
|------|------|------|
| ⓪ 环境检测 | 自动检测网络、iCloud、`~/.openclaw/`、磁盘空间 | 自动 |
| ① 欢迎页 | 产品价值展示，3 个卖点轮播 | — |
| ② 连接 AI 账号 | Claude采用setup-token/API key，ChatGPT采用OAuth/API Key，MiniMax采用OAuth，Kimi采用OAuth | ✅ |
| ③ 认识你的助手 | Agent 名字、emoji、用户昵称、语言偏好 | ✅ |
| ④ 解锁更多能力 | 文件/日历/提醒事项权限授权 | 可跳过 |
| ⑤ 初始化 | 启动 Gateway，5-10 秒加载动画 | 自动 |
| ⑥ 首次对话 | Agent 打招呼 + 提示气泡 | 自动 |

**设计原则：**
- 全程不出现"终端"、"Node.js"、"Gateway"等技术词汇
- OAuth 失败时有 API Key 兜底
- 权限全部可选，跳过后首次触发时再引导

#### 3.1.2 老用户数据迁移

| 需求 | 说明 |
|------|------|
| 自动检测 | 启动时检测 `~/.openclaw/` 是否存在 |
| 数据预览 | 展示对话条数、记忆文件数、Agent 数量、占用空间 |
| 拷贝迁移 | 完整拷贝到 `~/Library/Application Support/ClawTower/`，原数据保留 |
| 进度展示 | 逐项显示迁移状态（✅ ⏳ ○） |
| 配置转换 | 读取 config.yaml，保留用户配置，覆盖 ClawTower 专属配置 |
| 凭证迁移 | API key/token 提取后存入 Keychain |
| 冲突检测 | 检测 OpenClaw Gateway 是否在运行，提示先关闭 |
| 身份继承 | 迁移后自动读取 IDENTITY.md/USER.md，跳过起名步骤 |

## 4. 非功能需求

### 4.1 性能

| 指标 | 要求 |
|------|------|
| App 启动到可用 | < 5 秒（含 Gateway 启动） |
| 消息发送到首字输出 | < 2 秒（取决于 AI 模型响应） |
| 内存占用 | Swift 主进程 < 100MB，Gateway < 300MB |
| CPU 空闲占用 | < 3% |
| 本地存储 | App Bundle ~80MB，数据目录按用户使用增长 |

### 4.2 安全

| 层面 | 措施 |
|------|------|
| API Token | Keychain 存储，不落盘明文，不入日志 |
| CloudKit 通信 | 端到端加密（AES-256-GCM） |
| 本地数据（macOS） | 依赖 FileVault 全盘加密 |
| 本地数据（iOS） | NSFileProtectionComplete |
| 进程通信 | localhost |
| 日志 | token/prompt/文件路径脱敏，debug/release 分级 |
| 诊断报告 | 发送前自动脱敏，用户确认后发送 |

### 4.3 可靠性

| 场景 | 处理 |
|------|------|
| Gateway 崩溃 | 指数退避自动重启，最多 3 次，之后提示用户 |
| 网络中断 | 消息排队，恢复后自动重发 |
| Token 过期 | App 内弹窗引导重新授权 |
| macOS 睡眠恢复 | Gateway 自动检测并重连 AI 服务 |
| CloudKit 不可用 | iOS 端消息本地缓存，恢复后同步 |

### 4.4 兼容性

| 平台 | 最低版本 |
|------|---------|
| macOS | 14.0 (Sonoma) |
| iOS | 17.0 |
| 芯片 | Apple Silicon (M1+) 优先，Intel 兼容 |

### 4.5 国际化

| 语言 | 阶段 |
|------|------|
| 中文（简体） | 首期 |
| English | 首期 |
